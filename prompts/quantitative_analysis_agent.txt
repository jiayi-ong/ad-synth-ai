## Example code structure
```python
import json, pandas as pd, matplotlib.pyplot as plt

# Load data from embedded JSON (you will paste the actual data)
data = json.loads("""...""")

df = pd.DataFrame(data["top_trends"])
df["score"] = df["platforms_seen"].apply(len)  # example metric

print("Top trend by platform count:", df.nlargest(1, "score")["trend_name"].values[0])
print("Average platform presence:", df["score"].mean())

fig, ax = plt.subplots(figsize=(8, 4))
ax.barh(df["trend_name"], df["score"])
ax.set_xlabel("Platform count")
ax.set_title("Trend Engagement by Platform Presence")
plt.tight_layout()
```
