### Step 3 — Build financial model with Python
MANDATORY: Call `execute_python` to compute a financial model with numpy. The code MUST:
1. Compute gross margin percentage across a range of price points (unit_cost × 1.1 to unit_cost × 8).
2. Compute break-even units at 3 candidate price points (assuming fixed monthly overhead = unit_cost × 500 as a conservative baseline if no better estimate is available).
3. Plot a line chart: x-axis = price_usd, y-axis = gross margin %; add horizontal dashed line at 40% benchmark; shade competitor price range if known.
4. Print key metrics to stdout.

CORRECT chart code pattern (do NOT call plt.close or save manually — the pipeline captures open figures automatically):
```python
import matplotlib
matplotlib.use('Agg')
import matplotlib.pyplot as plt
import numpy as np

unit_cost = 10.0  # REPLACE with actual unit_cost_usd from product_profile

prices = np.linspace(unit_cost * 1.1, unit_cost * 8, 200)
gross_margins = (prices - unit_cost) / prices * 100

# Candidate price points
candidates = [unit_cost * 2, unit_cost * 3.5, unit_cost * 5]
candidate_margins = [(p - unit_cost) / p * 100 for p in candidates]
fixed_costs = unit_cost * 500
break_evens = [int(fixed_costs / (p - unit_cost)) for p in candidates]

fig, ax = plt.subplots(figsize=(8, 5))
ax.plot(prices, gross_margins, color='steelblue', linewidth=2, label='Gross margin %')
ax.axhline(40, color='gray', linestyle='--', alpha=0.7, label='40% benchmark')
for p, m in zip(candidates, candidate_margins):
    ax.axvline(p, color='orange', linestyle=':', alpha=0.6)
    ax.annotate(f'${p:.2f}\n{m:.0f}%', xy=(p, m), fontsize=8, ha='center')
ax.set_xlabel('Price (USD)')
ax.set_ylabel('Gross Margin (%)')
ax.set_title('Price vs. Gross Margin Analysis')
ax.legend()
plt.tight_layout()
# Do NOT call plt.close() or plt.savefig() — pipeline captures open figures automatically.
for i, (p, m, be) in enumerate(zip(candidates, candidate_margins, break_evens)):
    print(f"Candidate {i+1}: ${p:.2f} -- {m:.1f}% margin, {be:d} units break-even")
```
